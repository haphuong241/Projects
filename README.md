## 📌 Introduction
This project demonstrates an end-to-end Power BI workflow, from extracting data from PostgreSQL to cleaning and transforming the dataset using Power Query (M Query), and finally building an interactive dashboard for analysis.

The project highlights practical skills in data extraction, data preparation, and business intelligence reporting.
