## 📌 Overview
This project presents an interactive **Insurance Business Dashboard** built with Power BI to analyze key performance indicators such as revenue, customer behavior, and KPI achievement.
The dashboard provides both **descriptive insights** and **AI-oriented analysis**, supporting data-driven decision-making in business operations.

---

## 🎯 Objectives
- Monitor overall **revenue performance**
- Evaluate **KPI achievement rate**
- Analyze **customer purchasing patterns**
- Identify trends and performance drivers
- Introduce **AI-driven analytical thinking** into reporting
  
---

## 📂 Dataset
The dataset includes:
- Sales transactions
- Customer information
- Time dimension (Year, Month)
- Product-related data

Data is structured using a **dimensional model (star schema)** for efficient analysis.

---

## 📈 Key Features
- Revenue tracking over time
- KPI performance monitoring
- Time-based trend analysis
- Interactive filtering and drill-down
- Clear business-oriented visualizations

---

## 🤖 AI-Oriented Insights
This dashboard incorporates **AI-oriented thinking** to move beyond descriptive analytics toward predictive and prescriptive insights:

- **Revenue Forecasting:**  
  Uses built-in time-series forecasting in Power BI to estimate future revenue trends and support business planning.

- **Customer Retention Prediction (Conceptual):**  
  Suggests leveraging historical transaction behavior to predict the likelihood of customer renewal.

- **Cross-selling Recommendation (Conceptual):**  
  Proposes using purchase patterns to recommend related products and increase revenue per customer.

- **Anomaly Detection (Conceptual):**  
  Highlights the potential use of AI to detect abnormal changes in revenue or KPI for risk monitoring.

---

## 📊 Tools & Technologies

- Data modeling (Star Schema)
- Built-in AI features (Forecasting)
